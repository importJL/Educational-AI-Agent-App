# Plan to Fix Dark/Light Mode Icon Alignment

## Issue
The dark/light mode icon in the navigation bar is displaced and not horizontally centered with the tab links and labels.

## Root Cause
After examining the code in `client/src/App.tsx`, the navigation bar uses a `justify-between` layout:
- Brand logo on the left
- Theme toggle button and tab buttons grouped together on the right

The theme toggle button has different styling (padding, height) compared to the tab buttons, causing misalignment.

## Solution
Modify the theme toggle button to match the tab buttons' styling for consistent alignment:

1. Set height to 36px (matching tab buttons)
2. Set horizontal padding to 16px (matching tab buttons)
3. Add border-radius of 4px (matching tab buttons)
4. Keep transparent background and white text color
5. Maintain the existing gap between theme toggle and tab buttons

## Implementation Steps
1. Edit `client/src/App.tsx`
2. Locate the theme toggle button (lines 54-60)
3. Update the button's style object to include:
   - height: 36
   - padding: "0 16px"
   - borderRadius: 4
   - background: "transparent"
4. Keep existing color: "#fff"

## Expected Result
The theme toggle button will have identical dimensions and spacing as the tab buttons, causing the icon to appear horizontally centered with the tab links and labels.
